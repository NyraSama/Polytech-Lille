In order to test the project you have to :

 - Make sure you are using a bash
 - Make sure you have the package gcc to compile c programs
 - In this directory type the following command in your bash :	Make
 - Now you can test by using this command : ./main {n} < {file}
	{n} : number of start (if you don't know, type nothing, the program will choose the start)
	{file} : the test file between the three file that are like "cvrp*"

This program use algorithm of the combinatory theory to find the best distribution for a delivery which need to satisfy some customers.
