In order to test the project you have to :

 - Make sure you are using a bash
 - Make sure you have the package gcc to compile c programs
 - In this directory type the following command in your bash :	gcc -o main main.c
 - Now you can test by using this command : ./main < {file}
	{file} : the test file between the file which are in the tests directory
 - You can also play by yourself, enjoy !

This program use algorithm to build a game of Hasami Shogi.
